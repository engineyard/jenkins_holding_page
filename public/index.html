Ok, you have booted an Engine Yard AppCloud instance. You are one step away from hosting your own Hudson CI.

Set up Hudson CI server
gem install engineyard-hudson
ey-hudson install_server

Options

The install_server task assumes that this environment is called hudson or hudson_server and that there is only one such environment.

If it cannot find an environment with these names, or there are multiple environments with these names (you might be using the new collaborator beta feature), then you will asked to provide more options.

Environment flags
ey-hudson install_server --environment hudson_env --account my_hudson_account

Finished?

When this command is complete, refresh this page to see Hudson CI hosted on AppCloud!
